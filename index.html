BookLib

BookLib is an interactive web application designed to manage an online book library. Built with HTML, CSS, JavaScript, and a SQL backend, it allows users to add, view, and delete book entries effortlessly. The project features a user-friendly interface with options to categorize books by genre, author, or publication date. Data persistence is achieved through a SQL database, ensuring that book information remains available even after a page reload. BookLib is ideal for personal use or as a community resource for book enthusiasts.

Nutrition Tracker - React Web App

Nutrition Tracker is an innovative React-based web application that helps users monitor their daily nutritional intake and make healthier food choices. The app provides a beautiful interface with categories like fruits, vegetables, and various foods, each offering detailed information on nutritional values, optimal consumption times, and preparation methods. Integrated with a nutrition API, the app ensures accurate and real-time data. It also features a unique image-based product name identification tool, allowing users to scan food items with a camera or upload images to instantly receive detailed information about the product. With an enhanced dashboard, a food scanner, and a comprehensive search functionality, Nutrition Tracker makes healthy eating both easy and enjoyable.
